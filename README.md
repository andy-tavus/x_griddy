# Griddy

Open `index.html` in a browser — no server.

## Play

1. **Setup** — Image first for character, goal, and obstacle (file picker). One fallback emoji each if no image. Place start/goal/obstacles on the grid → **DONE SETUP**.
2. **Moves** — One line per move (`3 east`, `2 south`, …). **GO** runs them.
3. Trail colors: N blue · S red · E yellow · W purple.
4. After win/fail: **Play again** (same puzzle) or **Setup** (edit).

## Images

Save from image search → **Choose file**. Stored in-page (data URL); no URLs from the web. Big uploads are auto-downscaled. Obstacle image: **Obstacle** mode → two clicks = rectangle.
